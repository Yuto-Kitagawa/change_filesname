# 用途
Windowsでまとめてファイルの名前を変えたいとき。
まとめて名前を変更すると、  
Sample（1）jpg  
Sample（2）jpg  
Sample（3）jpg  
Sample（4）jpg  
Sample（5）jpg  
Sample（6）jpg  
Sample（7）jpg  
Sample（8）jpg  
と、なると思います。()いらないですよね？
1,2枚なら普通に名前を変更したほうが早いけど、10枚くらいになってくるとめんどくさいよね。
そういう時に使ってください。

# 使い方
pythonのファイルがあるフォルダーと同じ場所に「file」というフォルダーを作成して下さい。  
あとはpythonを実行するだけです。  
```
python main.py
```
*※PythonをインストールするかGoogle Colabolatoryで実行してください。*  
*※実行前にディレクトリに移動するのを忘れないで。*
